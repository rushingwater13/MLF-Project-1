In the data folder 3 different folders called graph, raw data, confusion matrices. Each contains either .txt or .png files.
The raw data come in .txt files that were generated after running the ANN for a specified dataset. They are named based on the dataset that the ANN was tested on. 
For example running the Ann on the Gaussian 2D Wide data set would correspond to the g_2_w.txt.
Likewise .png files follow the same naming structure with distinctions for confusion matrices and graphs.
